# 安装步骤

1. [根据官网](https://www.emotiv.com/emotiv-launcher/)点击下载链接
2. 选择database folder和安装文件夹
3. 选择组件: 免费版下载"Emotiv launche"r和"EmotivBCI"
4. 安装, 在安装的同时浏览信息
